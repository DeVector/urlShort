Sistema Encurtado de URL Implementando um ecurtado de URL

Olá! Seja bem vindo ao sistema Encurtado de URL desenvolvido com o intuito de fazer um encurtar URLs.
Esse é um sistema para gerenciamento de urls simulando um simples encurtador de url desenvolvido em Angular e Spring Boot na versão 2.3.12.RELEASE.
O sistema está rodando localmente e tem como principal foco receber uma url padrão e gerar uma encurtada tendo assim uma maior facilidade para compartilhar.
O sistema pode ser realizados testes localmente em sua maquina, caso encontre alguma melhoria que possa ser aplicada a ele nos informe no Issues no GitHub aqui.
