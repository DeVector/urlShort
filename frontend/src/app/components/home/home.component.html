Sistema Encurtado de URL Implementando um ecurtado de URL

Olá! Seja bem vindo ao sistema Encurtado de URL desenvolvido com o intuito de fazer um mini sistema para encurtar URLs.
Esse é um sistema para gerenciamento de urls simulando um simples encurtador de url desenvolvido em Angular e Spring Boot 2.
O sistema está rodando localmente.
O sistema tem como principal foco realizar uma url padrão e gerar uma encurtada que de para o usuario ter uma maior facilidade para compartilhar.

O sistema está em processo de desenvolvimento e encontra-se falhas, porém estamos nos adequando para que seja feita a correção de falhas com o passar do tempo.
